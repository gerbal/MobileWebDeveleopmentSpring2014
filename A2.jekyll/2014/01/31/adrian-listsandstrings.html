7. Write a function to count how many odd numbers are in a list.

def count_odd(list):
    #Make a counting variable
    odd_count = 0
    for number in list:
        #Is it odd?
        if number % 2 == 1:
            odd_count += 1
    return odd_count

print (count_odd([1, 3, 5, 4]))

Output

3

8. Sum up all the even numbers in a list.

def sum_even(list):
    sum = 0
    for i in list:
        if i % 2 == 0:
            sum = sum + i
    return sum

print (sum_even([2, 8, 7, 1000]))

Output

1010
